# MVP Scope

## Included

- React/Vite editor
- C++/WASM deterministic map generation
- 2D terrain view
- height map view
- side view
- recipe save/load through Spring Boot
- PostgreSQL persistence
- basic World Instance data model
- basic 2D player/entity dots later in MVP
- Elasticsearch public search after publish flow exists

## Excluded

- combat
- multiplayer
- full 3D before core map/editor works
- server-side map generation
- server-side real-time simulation
- advanced auth before core persistence
- paid marketplace, comments, moderation
