# Document Pack Manifest

This pack defines a fresh World Forge project using:

- React/Vite frontend
- C++/WebAssembly browser map engine
- Spring Boot API
- PostgreSQL source of truth
- Elasticsearch search projection
- World Instance client-side simulation with server-side persistence
